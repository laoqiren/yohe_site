yohe

just a static blog generator, powered by Node.js

中文文档

Features

  • generating static blog simply.
  • local server to preview the blog.
  • Customizable information and theme.
  • suppoert tags,categories,pages,archives,custom pages,etc.
  • discuss,analysis,etc.

Installation

$ npm install yohe -g

Quick Start

Get help

$ yohe --help

Setup your blog

$ yohe init blog
$ cd blog

Create a new post

$ yohe new cluster

Generate static files

$ yohe build

Preview the blog

$ yohe server
